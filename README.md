# labo-7
¿Cuál es la diferencia entre autenticación y autorizacion?
La autenticación verifica la identidad del usuario, mientras que la autorización define los permisos o accesos que ese usuario tiene dentro del sistema.

¿Cuál es la función del token JWT en la guía?
La función del token JWT en la guía es mantener la sesión del usuario autenticado 
y permitir el acceso a las rutas o recursos protegidos del sistema sin necesidad de volver a iniciar sesión cada vez.
